# Marketing-Performance-Dashboard
A data analytics dashboard for tracking and evaluating marketing performance across advertising channels and campaigns. The dashboard analyzes key metrics such as impressions, clicks, conversions, and campaign effectiveness to support data-driven marketing decisions.
# Marketing Performance Dashboard

## Project Overview
This project presents a **Marketing Performance Dashboard** that analyzes advertising performance across four years (2017–2020).

The dashboard provides an overview of key marketing metrics including:

- Impressions (ad reach)
- Clicks (user engagement)
- Cost Per Mille (CPM)
- Cost Per Click (CPC)
- Average Pages Visited
- Average Time Spent on Website

The objective of this dashboard is to help marketing teams **monitor campaign performance and evaluate trends in advertising effectiveness over time**.

---

## Dashboard Scope

The first section of the dashboard provides an **overview of marketing performance from 2017 to 2020**, allowing users to compare key metrics across different years.

The main indicators include:

| Metric | Description |
|------|------|
| Impressions | Total number of times advertisements were displayed |
| Clicks | Number of times users clicked on advertisements |
| CPM | Cost per thousand impressions |
| CPC | Cost per click |
| Average Pages Visited | Average number of pages visited per session |
| Average Time on Website | Average time users spend on the website |

Users can select different **years (2017–2020)** to view how marketing performance changes over time.

---

## Key Trends (2017–2020)

### Advertising Reach Growth
The total number of **impressions increased steadily from 24.5M in 2017 to around 30M in 2020**, showing that marketing campaigns expanded their reach significantly.

### Increasing User Engagement
The number of **clicks increased from approximately 966K in 2017 to over 1M in later years**, indicating improved audience interaction with advertisements.

### Rising Advertising Costs
Advertising costs such as **CPM and CPC increased each year**, which may reflect increased competition in digital advertising platforms.

### Stable Website Engagement
Website engagement remained relatively stable during the four-year period:

- Average pages visited: around **3.5 pages**
- Average time spent on website: around **4 minutes**
